## Overview
VroomVault is a modern website designed to showcase a car-selling business. It provides users with a clear and structured way to explore vehicles, learn about the VroomVault workshop, and connect with the business.

The project emphasizes simplicity, clean design, and an intuitive user experience.

---

## Features
- Display of cars available through the business  
- Information about the VroomVault workshop and services  
- A section for users to describe a car they want to sell  
- Simple contact and connection interface  
- Clean and responsive layout
